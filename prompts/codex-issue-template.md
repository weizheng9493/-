# Codex Issue 提示词模板

当你希望 Codex 基于 GitHub Issue 执行任务时，可以复制下面模板。

```md
## 背景

请说明为什么要做这件事。

## 目标

请说明希望最终得到什么结果。

## 输入材料

- 

## 具体要求

1. 
2. 
3. 

## 验收标准

1. 
2. 
3. 

## 边界和风险

- 不要修改：
- 需要特别注意：
- 不确定内容请标注：

## 给 Codex 的执行指令

@codex 请基于这个 Issue 创建实现方案并提交 PR。要求：

- 遵守仓库 `AGENTS.md`。
- 保持修改范围收敛。
- 说明改动点、验证方式和风险。
- 如果信息不足，先在 Issue 中提问，不要直接猜测。
```
