# 产品概览

## 定位

航旅 AI PM 工作流是一个面向团队协作的工作流产品仓库，用于沉淀航旅场景下的问题发现、诊断、竞品分析、PRD、设计交付、前端实现和验收等 AI 辅助产品流程。

## 目标用户

- 产品经理。
- 设计师。
- 前端工程师。
- 业务分析和运营同学。
- 使用 Codex 或其他智能体完成产物生成、分析和实现的团队成员。

## 核心资产

| 资产 | 说明 |
| --- | --- |
| Agent Skills | 可被智能体复用的专项能力 |
| Workflows | 团队标准工作流 |
| Prompts | 可复用提示词 |
| Docs | 正式说明文档 |
| Artifacts | 团队成员或智能体生成的待沉淀产物 |
| Examples | 示例材料 |

## 产品化管理方式

本仓库按产品版本管理：

- 每次更新通过 PR 进入。
- 每次 PR 必须说明更新点和风险。
- 重要更新需要 reviewer 审核。
- 正式版本通过 `CHANGELOG.md` 和 GitHub Release 发布。

## 当前阶段

当前处于初始化阶段，优先建立：

- 仓库目录结构。
- PR 和 Issue 模板。
- 智能体产物提交规范。
- 发布流程。
- 分支保护规则。
